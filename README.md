# Traffic Light Junction System

## Description
A two-road traffic signal system built using Arduino and 6 LEDs in Tinkercad.

## Components Used
- Arduino Uno
- 6 LEDs
- 6 Resistors
- Breadboard
- Jumper Wires

## Software Used
- Tinkercad

## Working
Road 1 gets green signal while Road 2 remains red.
Then Road 1 changes to yellow.
Next Road 2 gets green signal while Road 1 remains red.
The cycle repeats continuously.
