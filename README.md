# Traffic Light Junction System

## Overview
This project simulates a two-road traffic signal junction using Arduino Uno and 6 LEDs in Tinkercad.

## Components Used
- Arduino Uno
- 6 LEDs (2 Red, 2 Yellow, 2 Green)
- 6 Resistors (220Ω)
- Breadboard
- Jumper Wires

## Software Used
- Tinkercad
- Arduino IDE
- GitHub

## Circuit Design
The circuit consists of two traffic roads. Each road has Red, Yellow, and Green LEDs connected to Arduino digital pins.

## Working Principle
1. Road 1 Green LED turns ON while Road 2 Red LED remains ON.
2. Road 1 Yellow LED turns ON before switching.
3. Road 2 Green LED turns ON while Road 1 Red LED remains ON.
4. The cycle repeats continuously.

## Embedded C Concepts Used
- Variables
- Functions
- GPIO Output
- Digital Write
- Delay Function
- Program Flow Control
